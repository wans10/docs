LLM Hub 帮助开发者获取和优化 AI 应用。我们相信未来是多模型和多提供商的时代。

## 为什么选择 LLM Hub ？

**价格与性能**: LLM Hub 在数十家提供商中寻找最优价格、最低延迟和最高吞吐量的提供商，并允许您选择如何[优先排序](/docs/features/provider-routing)。

**标准化 API**: 在模型或提供商之间切换时无需更改代码。您甚至可以让您的用户[选择并支付他们自己的费用](/docs/use-cases/oauth-pkce)。

**真实世界洞察**: 率先利用新模型。查看不同用途的[模型使用频率](https://openrouter.ai/rankings)的真实数据。关注我们的[Discord频道](https://discord.com/channels/1091220969173028894/1094454198688546826)，了解最新动态。

**整合计费**: 无论您使用多少个提供商，计费都简单透明。

**更高的可用性**: 后备提供商和自动智能路由意味着即使提供商出现故障，您的请求仍然有效。

**更高的速率限制**: LLM Hub 直接与提供商合作，提供更优的速率限制和更高的吞吐量。
